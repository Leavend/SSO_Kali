# Self-hosted Fonts

These fonts are self-hosted to avoid the render-blocking handshake
with fonts.googleapis.com / fonts.gstatic.com (ISS-PERF3).

## Instrument Serif (regular 400)

- File: `instrument-serif-regular.woff2`
- Size: ~15 KB
- Source: Google Fonts — Instrument Serif v5
  (https://fonts.google.com/specimen/Instrument+Serif)
- License: SIL Open Font License 1.1
  (https://openfontlicense.org/open-font-license-official-text)
- Latin subset only. Italic and extended-Latin variants are
  intentionally omitted for first-paint budget; add as needed.
